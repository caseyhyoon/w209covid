Description

Two sections on this dashboard Map Cases by Month and Calendar by State that show views of newly confirmed cases within the US by state and time.
Timeframe: January 2020 - February 2021

Map Cases by Month
  • Select a month, shows daily calendar view of new confirmed cases on the left and map view on the right for that specific month
  • Click on a state on the map for data specific to the state

    • NOTE: Only select a state after selecting a month

  • Click on the blank space to view new confirmed cases for the entire US

Calendar by State
  • Select a state, shows calendar view of new confirmed cases for that specific state
  • Hover over day for specific details
